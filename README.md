# Python RAG, AI para los PDFs

Proyecto base para probar un sistema RAG con documentos PDF.

## Estructura

- `data/`: PDFs de ejemplo.
- `backup/`: copias de seguridad.
- `chroma/`: almacenamiento de base vectorial.
- `__pycache__/`: carpeta replicada para mantener la estructura del proyecto de referencia.
- `.pytest_cache/`: carpeta replicada para mantener la estructura del proyecto de referencia.
